Database for displaying cats:

cd /www
    rails new feline
cd feline
    rake db:create
    sudo bundle install
    rails generate controller home index
edit config/routes.rb ---->added resources :cats
    rails generate scaffold Cat name:string img:string
    rake db:migrate
copied cat images to public/images/ - ex cat1.jpeg
